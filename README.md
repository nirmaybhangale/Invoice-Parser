This AI Document Intelligence Engine automates the transition from raw documents to structured data by combining EasyOCR and pdfplumber with Llama-3 via the Hugging Face Router. The pipeline intelligently extracts text from both digital and scanned PDFs or images and uses advanced LLM prompting to distill complex information into clean, actionable insights. Delivered through a responsive Streamlit interface, it offers a modular and scalable solution for high-accuracy document analysis and data extraction.

Type "streamlit run app.py" to run the app locally.
